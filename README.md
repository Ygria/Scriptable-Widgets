# 脚本说明


## Golden Widget 实时获取伦敦交易金价

注意：周末非交易时间无法获取。

>调用接口获取伦敦金价。

## 如何下载

点击进本仓库的scriptable 文件夹，点击进脚本文件（`gold_widget.js`），点击右上角下载图标-下载raw file

- 手机端，点右上角三个点，然后点Download，允许下载即可
